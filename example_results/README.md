# Example metrics (reference only)

These JSON files are **illustrative snapshots** from one recorded benchmark run (aggregate summaries and run flags only).
They are **not** raw CGM traces and contain **no** steward XML.

Reproduce your own metrics with:

```bash
export GLUCOTWIN_EXPERIMENT_ONLY=1
PYTHONPATH=. python3 src/experiments/run_all.py
```

Your own `outputs/results/` after a full run is the authoritative record for any quantitative claim.
